The Data YouTubers Dashboard

2025 Persona: Fabric Analytics Engineering

Overview

Built a comprehensive public dashboard that provides insights into the data analytics YouTube community. The dashboard has been viewed by over 30,000 people in the data community and has helped 150+ YouTube creators benchmark their performance against peers in the data industry.

Key Features

  • Public dashboard with 30,000+ viewers in the data community
  • Performance benchmarking for 150+ YouTube creators
  • Discovery platform for learners to find new data content creators
  • Daily refreshed metrics and insights

Technical Implementation

Technologies Used

  • Fabric notebook and YouTube Data API for data collection
  • Fabric Lakehoue and warehouse as storage in medallion architecture
  • Fabric warehouse and stored procedures for modeling
  • Fabric Pipeline for orchestration and daily data refresh
  • Power BI for visualization and dashboard creation
