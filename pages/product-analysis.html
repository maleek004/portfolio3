Product Analysis Dashboard

2025 Persona: Product Analyst

Overview

The fictional product team at Vendbridge needed better visibility into app performance and user behavior across different regions, months, and devices to improve user experience and drive growth.

Built an interactive dashboard that tracks total app installs, user signups, signup rate, churn rate, average daily crashes, uninstalls, daily active users, average time spent, and total transactions. All filterable by month, region, and device, and compared key metrics month-over-month.

Key Features

  • Comprehensive app performance metrics tracking (installs, signups, crashes, uninstalls)
  • User engagement analytics (daily active users, average time spent)
  • Business metrics monitoring (signup rate, churn rate, total transactions)
  • Multi-dimensional filtering by month, region, and device
  • Month-over-month comparison capabilities for trend analysis

Technical Implementation

Technologies Used

  • Google sheet for pre-analysis
  • Power BI for interactive visualization and dashboard creation
